# Philip Luo — Portfolio

A single-file, deep-space themed personal portfolio. The entire scene — drifting
constellations, a glowing sun with orbiting planets, a bumpable astronaut, and
periodic depth-aware asteroid showers — is hand-coded in plain JavaScript and
HTML5 canvas. No images, no libraries, no external assets.

## What it is

An interactive night-sky landing page with an expanding glass-card menu
(about · experience · projects · skills · contact). Built to be visually
distinctive while staying substance-first for recruiters: the animated scene is
the front door, and the real content sits one tap behind it.

## Stack

- Plain HTML, CSS, and vanilla JavaScript — one self-contained `index.html`
- HTML5 canvas for the generative space scene
- Google Fonts (Sora) for typography
- Zero build step, zero dependencies

## Run it

Open `index.html` in any modern browser. That's it.

## Deploy it

- **Netlify** — drag the file onto [Netlify Drop](https://app.netlify.com/drop)
- **GitHub Pages** — push to a `yoman321.github.io` repo and enable Pages

## Accessibility

Respects `prefers-reduced-motion` (the scene settles into a calm static state),
uses `aria-expanded` on the menu, and keeps visible keyboard focus rings.
